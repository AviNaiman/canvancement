Quick Start

  1. Choose Configure API Settings and enter your Canvas Instance and your Access Token.
  2. Choose Specify Course and put in the Canvas Course ID for the course to analyze.
  3. Choose Refresh All Discussions to download and analyze your data. Note that this routine will erase any existing data and do a complete fetch of all of the discussions.

The Append New Discussions will fetch discussions that are not already in the data table. Use this option if you want to run this program multiple times throughout the term and do not want to waste time re-downloading exisiting information. Note that if you choose this option, then previously counted discussions will not be checked for new posts.

Understanding the Report

After counting the posts, you will have a report in the form of a pivot table.

The rows will contain the names of the students and the columns will contain the discussion titles. Both will be in alphabetical order.

By default, results are shown only for active students. You may use the filters in the report dialog on the right hand side to include completed students or teacher or ta roles.
